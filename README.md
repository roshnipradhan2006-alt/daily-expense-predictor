# AI Daily Expense Predictor

## Project Overview
This project predicts a person's daily expenses using machine learning.

The model uses factors such as income, previous expenses, and whether the day is a weekday or weekend to estimate how much a person might spend.

## Technologies Used
Python
Pandas
Scikit-learn

## Machine Learning Model
Linear Regression

## Dataset
The dataset contains:
Income
Day type
Previous expense
Actual expense

## How to Run
1. Install Python
2. Install libraries
pip install pandas scikit-learn

3. Run the Python file
python expense_predictor.py

## Future Improvements
More features like lifestyle habits and location-based expenses can improve predictions.
